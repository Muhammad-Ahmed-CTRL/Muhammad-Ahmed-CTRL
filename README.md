# 💫 About Me:
🎓 **Computer Science Student** specializing in low-level systems programming, cybersecurity automation, cross-platform mobile ecosystems, and advanced algorithmic engine design.

⚙️ **Low-Level Systems & OS Development:** I write bare-metal code. Deeply engaged in building custom operating systems from scratch using **NASM x86 Assembly**, implementing cooperative multitasking kernels, hardware drivers, and custom shell parsers.

🛡️ **Cybersecurity & Static Analysis:** Passionate about security automation. I build advanced vulnerability scanners, real-time static code analysis engines, deep Git forensics tools, and automated web crawling/assessment platforms targeting vulnerabilities like XSS, SQLi, and RCE.

📱 **Mobile Architecture & Full-Stack Hubs:** Expert in the **Flutter & Dart** ecosystem integrated with **Firebase** backends—architecting complex, multi-module applications (like `Edu-Verse`) supporting real-time student-admin marketplaces, recruitment, and AI planner integrations.

🌐 **Web & Freelancer Intelligence:** Building modern, analytical web tools using **Next.js** and **TypeScript** to deliver complex processing, such as automated tax calculations, government compliance exemptions, and AI-powered client workflow systems.

🧠 **Algorithms & Simulation Engines:** High-performance engine design in **C++** utilizing **SFML** to visualize advanced spatial partitioning systems (QuadTrees) and complex pathfinding algorithms.

🚀 **Future Horizons:** Transitioning gears into **Agentic AI Orchestration** and advanced **Data Engineering** pipelines to build self-sustaining, intelligent software ecosystems.
